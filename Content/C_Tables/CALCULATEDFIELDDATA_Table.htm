CALCULATEDFIELDDATA Table

The CalculatedFieldData table contains the standard calculated fields included with Saleslogix, as well as those created for custom use. In Saleslogix, calculated fields are defined as several fields from one or more tables that are treated as a single unit. For example, character fields may be combined, or mathematical formulas may be used to create calculated fields.
