CAMPAIGNSTAGE Table

Added in Version 7.0

The CAMPAIGNSTAGE table stores all the information pertaining to the stages of a campaign. This information includes, but is not limited to, campaign stage ID, campaign ID, status, description, lead source ID, campaign code, stage start date, stage end date, launch date, and notes.
