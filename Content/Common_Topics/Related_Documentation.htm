Related Documentation

In addition to this online Help system, the following printed and online documentation is available:

  • The Infor CRM Implementation Guide  provides step-by-step instructions for installing Infor CRM. The Infor CRM Implementation Guide is available on the Infor CRM media.
  • The Administrator Help provides instructions for configuring and managing users and the Infor CRM system. In addition, you can create custom tables and fields using the Database Manager. Help is available by pressing F1 in the Administrator.
  • The Query Builder Help provides instructions to enable you to select data from the database based on one or more conditions. You can use this data to manage lookups and groups, subscription rules, and realign territories. Query Builder Help is available from the Administrator, Architect, and Infor CRM Client applications.
