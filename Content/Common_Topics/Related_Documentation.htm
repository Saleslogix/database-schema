Related Documentation

In addition to this online Help system, the following printed and online documentation is available:

  • The Saleslogix Implementation Guide provides step-by-step instructions for installing Saleslogix in a Windows and Web environment. The Saleslogix Implementation Guide is available on the Saleslogix DVD.

  • The Administrator Help provides instructions for configuring and managing users and the Saleslogix system. In addition, you can create custom tables and fields using the Database Manager. Help is available by pressing F1 in the Administrator.

  • The Query Builder Help provides instructions to enable you to select data from the database based on one or more conditions. You can use this data to manage lookups and groups, subscription rules, and realign territories. Query Builder Help is available from the Administrator, Architect, and Saleslogix Client applications.
