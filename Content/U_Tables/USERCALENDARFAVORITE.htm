USERCALENDARFAVORITE Table

Added in Version 8.1

The USERCALENDARFAVORITE table stores the list of favorite users that appear in the user calendar.

  • While this table does not have a field defined as a primary key, the USERCALENDARFAVORITEIID field should be unique and considered as a primary key.

  • CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
