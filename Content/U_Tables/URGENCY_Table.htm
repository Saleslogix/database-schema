URGENCY Table

The URGENCY table stores information about urgency codes that are used to prioritize tickets in Saleslogix. Urgency codes indicate the priority of the ticket and how immediately to respond to the ticket. The information includes a numeric urgency code, description, turnaround time in minutes, and settings to enable the urgency code and allow Web Ticket user access.
