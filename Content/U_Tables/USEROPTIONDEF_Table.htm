USEROPTIONDEF Table

Added in Version 6.2

The USEROPTIONDEF table stores options, definitions, and default values. In order for a value to exist in UserOptions, a corresponding definition of the option must exist in the USEROPTIONDEF table.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
