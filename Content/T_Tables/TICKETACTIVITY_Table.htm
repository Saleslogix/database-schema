TICKETACTIVITY Table

The TICKETACTIVITY table stores detailed information on all support activities, such as On-site consulting, associated with tickets. An activity is an action or event that occurs during the processing and resolution of a ticket. This includes any process, action, step, or work done by a customer service representative or technical service representative to help solve a customers problem or resolve a reported defect. The information stored in this table includes the TicketID, rate type description associated to the ticket, activity type code, due date, assigned date, completed date, and other details.
