INTEGRATION Table

Added in Version 8.1

The INTEGRATION table maintains a list of third party applications that provide a means of synchronization with Saleslogix.

 

While this table does not have a field defined as a primary key, the INTEGRATIONID field should be unique and considered a primary key.
