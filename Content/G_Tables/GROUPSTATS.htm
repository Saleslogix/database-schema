GROUPSTATS

Added in Version 8.1.0.05

The GROUPSTATS table stores group usage statistics that can be used to manage large numbers of groups. For example, when a group plugin was last accessed and the number of times a group has been accessed.

Group statistics are updated using the TrackGroupUsage method in GroupBuilder. The method is called from the Groups SystemAdapter endpoint when a group response is sent back to the client.
