DB_OJECTDEFINITION Table

Added in Version 8.1

The DB_OBJECTDEFINITION table stores definitions of database-specific database objects such as Views and Triggers.

While this table does not have a field defined as a primary key, the DB_OBJECTDEFINTIONID  field should be unique and considered as a primary key.
