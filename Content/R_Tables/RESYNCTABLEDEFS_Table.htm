RESYNCTABLEDEFS Table

The RESYNCTABLEDEFS table stores information on every table in the database and is used by synchronization. The information includes the table name, key field, secondary field, secure and omni-directional attributes, the display name for the table, if the table is primary, and a description for the table.

The RESYNCTABLEDEFS table should not be modified unless directed by Saleslogix Technical Support. Changes to this table could cause irreversible data corruption.
