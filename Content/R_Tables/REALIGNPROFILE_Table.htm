REALIGNPROFILE Table

Added in Version 6.0

The REALIGNPROFILE table stores information created for territory realignment using the Administrator (Tools > Realign Territories).

Table relationships

The REALIGNPROFILE table has a (Many) to (Zero or One) relationship with the USERINFO table.

AccountManagerID, RegionalManagerID, DivisionalManagerID, CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
