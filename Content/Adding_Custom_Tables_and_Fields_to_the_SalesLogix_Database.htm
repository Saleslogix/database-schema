Adding Custom Tables and Fields to the Saleslogix Database

To add custom tables and fields to the Saleslogix database, Saleslogix recommends using the Database Manager. Tables and fields that are not added using the Database Manager are disabled; they are not integrated into security or synchronization, and appear in the Database Manager tree view with gray-colored icons. Any table or view added to the Saleslogix database from outside the Database Manager must be enabled.
