OAUTHPROVIDER Table

Added in Version 8.1

The OAUTHPROVIDER table stores the set of OAuth providers available to the system and to users. Added for integration functionality.

While this table does not have a field defined as a primary key, the OAUTHPROVIDERID field should be unique and considered as a primary key.
