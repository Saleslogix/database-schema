NETFORMS_PLUGIN Table

Added in Version 7.0

The NETFORMS_PLUGIN table stores all the information needed for managing net forms plugins. This table contains information including, but not limited to, the entry point ID, title, file name, source, version, data date and data.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
