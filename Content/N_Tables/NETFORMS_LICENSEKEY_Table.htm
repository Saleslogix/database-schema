NETFORMS_LICENSEKEY Table

Added in Version 7.0

The NETFORMS_LICENSEKEY table stores the license keys associated with specific add-on functionality. This table contains information including, but not limited to the plugin ID, identifier, description and data.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
