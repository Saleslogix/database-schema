LOOKUP Table

The LOOKUP table stores searches that are performed using the Lookup Manager. The information contains the main table name and search field, the ID for the field, the display name for the field searched, the layout of the search, pick list name for the field (if applicable) and the field type and lookup name.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
