LICENSES Table

The Licenses table contains license numbers assigned to users (Network, Remote and Web users), Administrator, synchronization servers, Remote Office servers, and support servers. The license number determines the type of license. Licenses are entered in the Administrator.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
