LITREQUEST Table

The LitRequest table contains information regarding the literature requested and fulfilled for a contact. Literature requests are scheduled using the Saleslogix Network or Web Client and include the requested date, send date, user requesting the literature, description of the request, how to send the request (mail, UPS, etc), the cover letter ID, name to send with the request, and the contact information.

Literature fulfillments are performed using the Saleslogix Client.
