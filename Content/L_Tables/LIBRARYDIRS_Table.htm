LIBRARYDIRS Table

The LibraryDirs table stores the location (path) in relationship to the "Library" folder. The path to the Library folder is stored in the BRANCHOPTIONS table. The information contains the directory ID, the ID of the parent folder, the folder name, full path for the folder, and if the folder is found. Saleslogix Library folders are created in the Administrator.
