META_CLASSFUNCTION Table

The META_CLASSFUNCTION table stores class and function information for SalesLogix classes and functions. This information includes the captions and hints that appear for Saleslogix actions.

Table Relationships

  • The META_CLASSFUNCTION table has a (Many) to (Zero or One) relationship with the META_CLASS table.

  • The META_CLASSFUNCTION table has a (Many) to (Zero or One) relationship with the META_FUNCTION table.

  • The META_CLASSFUNCTION table has a (Many) to (One) relationship with the SEC_FUNCTIONOWNER table.

  • CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.

  • If an index entry appears in multiple table columns, then it is a composite index. A composite index requires multiple columns in order to create a unique index.
