HIST_MKTGSVC_CLICK Table

Added in Version 7.0

The HIST_MKTGSVC_CLICK table stores all the history information collected when a target clicks a link within an e-mail from the marketing service. This table contains information including, but not limited to the lead's response ID, the campaign ID, the date a link was clicked, the name of the link, and the link URL.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
