ACTIVITYTRACKEVTHIST Table

Added in Version 7.5

The ACTIVITYTRACKEVTHIST table stores activity status event information for running processes. This table holds all of the status events for each activity, not just the most recent status event for each activity.

table relationships

The ACTIVITYTRACKEVTHIST table has a (Many) to (One) relationship with thetable.
