ACCOUNTPRODUCT Table

The ACCOUNTPRODUCT table stores information about products that are associated to an account.

table relationships

  • The ACCOUNTPRODUCT  table has a (Zero or One) to (Many) relationship with the CONTRACTITEM table.

  • The ACCOUNTPRODUCT  table has a (Many) to (One) relationship with the ACCOUNT and PRODUCT tables.

If an index entry appears in multiple table columns, then it is a composite index. A composite index requires multiple columns in order to create a unique index.
