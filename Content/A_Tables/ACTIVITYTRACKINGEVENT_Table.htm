ACTIVITYTRACKINGEVENT Table

Added in Version 7.5

The ACTIVITYTRACKINGEVENT table stores the current status of each activity that has had at least one event occur during process execution.

While this table does not have a field defined as a primary key, the ACTIVITYTRACKINGEVENTID field should be unique and considered a primary key.
