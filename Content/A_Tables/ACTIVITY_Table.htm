ACTIVITY Table

The Activity table stores information about meetings, phone calls, and to-dos that are scheduled for a user. The information includes activity type, account name, contact name, opportunity name, start date, duration, description, alarms, and other variables associated with a Saleslogix activity. The ACTIVITY table also contains several user-defined fields with the datatype of variable character and length of 255 characters.

Events are stored in the EVENT table and literature requests are stored in the LITREQUEST table.

If you applied the SLX Indexes for MS SQL.sql script to your database, your index information may be different than described in this topic.  For more information, see the Index Changes.pdf document.
