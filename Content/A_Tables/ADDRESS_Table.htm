ADDRESS Table

The ADDRESS table stores the address information for Saleslogix accounts, contacts, users, and Remote Offices. This information includes type of address, description, number and street, city, state, postal code, county, country, designations for use as primary or mailing address, salutation to use for mailing, routing, additional addresses, and time zone.
