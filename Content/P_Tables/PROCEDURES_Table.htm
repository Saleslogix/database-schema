PROCEDURES Table

Deprecated in Version 7.0

The PROCEDURES table stores information about procedures. Procedures are steps or processes for common tasks. Each procedure can have a set of steps, confidence level, several contributors and links to other files.

table relationships

The PROCEDURES table has a (Zero or One) to (Many) relationship with the PROCEDURECONTRIBUTOR table.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
