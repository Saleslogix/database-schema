PROCESSTRACKINGEVTHIST Table

Added in Version 7.5

The PROCESSTRACKINGEVHIST table stores process event information for running processes.  This table holds all events for a process, not just the most recent event.

table relationships

The PROCESSTRACKINGEVTHIST table has a (Many) to (One) relationship with the PROCESSINSTANCESTATUS table.
