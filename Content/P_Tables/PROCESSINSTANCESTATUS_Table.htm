PROCESSINSTANCESTATUS Table

Added in Version 7.5

The PROCESSINSTANCESTATUS table is the parent table for all process tracking.  The table contains one record for each instance of a running process.

While this table does not have a field defined as a primary key, the PROCESSINSTANCESTATUSID field should be unique and considered a primary key.
