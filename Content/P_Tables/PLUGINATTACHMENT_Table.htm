PLUGINATTACHMENT Table

Added in Version 6.0

The PLUGINATTACHMENT table stores information about attachments for the E-mail, Fax, and Letter template plugins used in Saleslogix Mail Merge.

Table Relationships

The PLUGINATTACHMENT table has a (Many) to (Zero or One) relationship with the PLUGIN table.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
