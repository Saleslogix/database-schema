PROCESSTYPECATEGORY Table

Added in Version 7.5

The PROCESSTYPECATEGORY table stores 0 or more categories that are associated with a process type.  This allows each process type to be grouped independently of its type.

While this table does not have a field defined as a primary key, the PROCESSTYPECATEGORYID field should be unique and considered a primary key.
