SLXQUERYPLANHINT Table

Added in Version 6.2.4

The SLXQUERYPLANHINT table stores information required for the SQL Optimization Toolkit. The query optimizer stores information about possible optimizations of corresponding ‘original’ queries. These are optimized queries or instructions for how to optimize original queries. This table contains information including, but not limited to, the Infor CRM query plan ID, hint type, data, database type and category.

Table Relationships

The SLXQUERYPLANHINT table has a (Many) to (One) relationship with the SLXQUERYPLAN table.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
