SITEKEYS Table

The SITEKEYS table is used to store site code information needed for synchronization. Saleslogix uses the Saleslogix key file to generate unique keys for data entered into the database. Keys are generated based on each user's database. This table is a child to the parent table, SITEOPTIONS.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
