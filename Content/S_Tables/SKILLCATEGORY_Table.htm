SKILLCATEGORY Table

Deprecated in version 7.0

The SKILLCATEGORY table stores information for a user's skills and experience level. For users of Support features, category and experience level items can be set in the Administrator (user profile > Support tab).

  • CategoryID and ExperienceCode are created using Pick List Manager.

  • CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
