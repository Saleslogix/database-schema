SECFUNCTIONS Table

Added in Version 6.2

The SECFUNCTIONS table stores information on Saleslogix Network and Remote Client functions and commands that can be invoked by menus or views. The information includes the function or view name, parent and program identification, the display name, the resource identification, the description of the function or view, and if the default is enabled.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
