SLXOLEDBPLUGIN Table

Added in Version 7.0

The SLXOLEDBPLUGIN table stores information related to SLX OLE DB plugins. This table contains information including, but not limited to, the description, name, the dll path, object type, and the primary dll. This information relates to defaults, scalar and custom security.

Table Relationships

  • The SLXOLEDBPLUGIN table has a (Many) to (One) relationship with the SLXOLEDBPLUGINDATA table.

  • The SLXOLEDBPLUGIN table has a (Zero or One) to (Many) relationship with the SLXSQLSCALARFN table.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
