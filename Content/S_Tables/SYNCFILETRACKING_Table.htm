SYNCFILETRACKING Table

The SYNCFILETRACKING table contains the name of a requested log file, source location, target location, and file status of each missing, requested, or recovered log file.

  • SourceSite and TargetSite are foreign keys referencing SiteCode in the SITEOPTIONS table.

  • CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
