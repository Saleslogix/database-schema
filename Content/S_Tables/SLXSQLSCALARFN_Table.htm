SLXSQLSCALARFN Table

Added in Version 7.0

The SLXSQLSCALARFN table stores configuration entries for scalar functions. These scalar functions are not real functions but rather a way to emulate a scalar function by changing queries on the fly. There several potential uses of these scalar functions. For example, they allow database agnostic syntax for DATE, STRING, and other functions.

Table Relationships

The SLXSQLSCALARFN table has a (Many) to (Zero or One) relationship with the SLXOLEDBPLUGIN table.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
