SLXWEBUSERDEVICE Table

The SLXWEBGROUPUSERDEVICE table stores information needed to handle the Web Phone automatic login feature.  This table contains information including, but not limited to, the user ID and unique phone device ID.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
