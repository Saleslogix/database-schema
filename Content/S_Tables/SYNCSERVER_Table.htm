SYNCSERVER Table

The SYNCSERVER table stores the Sync Server profile information. This information is entered using the Administrator and includes the synchronization server site code, shared server path for logging, the main office server site code, and other details. This table is a child to the parent table, SITEOPTIONS.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
