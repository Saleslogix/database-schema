SECTABLEDEFS Table

The SECTABLEDEFS table defines all tables and fields in the Saleslogix database There is one record for every field in every table. The information includes the table name, field name, the field offset number, the field index number (if applicable),  the sequence code and key flag, whether the field is a user-defined field and can it be synchronized to remotes, the display name, and whether or not the field is hidden (not displayed in Query Builder).

  • CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.

  • If an index entry appears in multiple table columns, then it is a composite index. A composite index requires multiple columns in order to create a unique index.
