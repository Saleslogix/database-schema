SYNCSERVICEFILE Table

Added in Version 6.0

The SYNCSERVICEFILE table stores the path to the SyncService.cfg file. The path information is entered using the Administrator (Systems > Sync Automation Services tab).

  • The scheduling information for the service configuration is stored in the .cfg file, not in the database.

  • CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
