SALESPROCESSAUDIT Table

Added in Version 6.2

The SALESPROCESSAUDIT table is a child table to SalesProcesses and stores sales process items.

While this table does not have a field defined as a primary key, the SALESPROCESSAUDITID field should be unique and considered a primary key.
