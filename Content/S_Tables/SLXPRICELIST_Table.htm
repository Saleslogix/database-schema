SLXPRICELIST Table

Added in version 7.5.4

The SLXPRICELIST table stores the grouping of commodities and their prices. The table is used for accounting integration functionality.

While this table does not have a field defined as a primary key, the SLXPRICELISTID field should be unique and considered a primary key.
