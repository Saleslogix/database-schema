SYNCSECTABLES Table

Added in Version 6.2.3

The SYNCSECTABLES table stores information used to identify tables that use SeccodeID for Sync control. This table contains information including, but not limited to, the table name and key field.

CreateUser and ModifyUser are foreign keys referencing UserID in the USERINFO table.
