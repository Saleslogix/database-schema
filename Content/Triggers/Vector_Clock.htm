Vector Clock Information

Updates the SYNCDIGEST Table and GLOBALCHANGETRACKING tables.

  • SYNCDIGEST table
    • Fetches the next tick value to be written to the GLOBALCHNAGETRACKING table. This is based on the Source(APPID) and Resource being updated. The tick value is then incremented and written back to the SYNCDIGEST.
  • GLOBALCHANGETRACKING table
    • Deletes existing records with matching ENTITYID and CHANGETYPE
    • Inserts new records
      • DISGESTTICK: The value retrieved from the SYNCDIGEST table
      • CHANGETYPE:
        • 0 – for an insert or update
        • 1 – for a delete
      • APPID: The Source APPID(SLX APPID if not set)
      • GLOBALSYNCID: The GLOBALSYNCID of the record changed
